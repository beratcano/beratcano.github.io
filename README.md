# beratcano.github.io

Personal portfolio website showcasing projects, skills, and contact information.

**Live:** [beratcan.me](https://beratcan.me)

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- GitHub Pages hosting
- No build process - static files

## Features

- Light/Dark theme toggle
- Responsive design
- Interactive skills compass chart
- Project showcase pages

## Projects Included

- **ShotGlow** - Screenshot beautifier with gradient backgrounds
- **Leaf Chroma** - Seasonal plant data visualization
- **League Hub** - Gaming statistics dashboard

## Local Development

```bash
# Start local server
python -m http.server 8000
# or
npx serve .
```

## Deploy

Push to `main` branch - GitHub Pages handles the rest.
