Why I built this

Just fun games to play with friends. No deeper purpose - sometimes you build things because they're enjoyable to make and use.

What it does

Gaming statistics dashboard with player profiles and interactive wheel spinner. Features player stats, game history, and a random champion selector wheel. We use it when we can't decide what to play.

Built with

  • Vanilla JavaScript
  • CSS Grid
  • HTML/CSS

What I learned

Building for fun is valid. Not everything needs to solve a business problem.
